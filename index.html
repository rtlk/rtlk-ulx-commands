Welcome to RTLK's ULX Commands.

This GitHub hosts a group of commands created for the ULX administration system. The current release has the following commands:

Pointshop (pointshop-ulx-commands):
!givepoints - Give's the target player(s) a number of points.
!takepoints - Takes the specified number of points from the target player(s)
!setpoints - Sets the target player(s) to have the specified number of points.
!giveitem - Gives the targeted player(s) the specified pointshop item (if they do not have it already.)

Cinema (cinema-ulx-commands):
!skipvideo (!skip) - Skips the current video.
!seekvideo (!seek) - Seeks to a certain point in the current video.
!resettheater (!reset) - Resets the theater.

Installation

To install one of the command packs in this repo, just copy the folder (ie. pointshop-ulx-commands) to your addons directory and restart the server!
